Developed a comprehensive e-commerce platform for managing and selling various categories of footwear online. Implemented an admin panel with login, CRUD operations for products, sales reports, and order management. Built a user-friendly interface with registration, login, product browsing, search, filtering, and cart management features. Integrated Razorpay for secure payment processing, order confirmations, and email receipts. Enabled QR code generation and scanning to enhance user experience, resulting in a streamlined platform for footwear sales and inventory management.
